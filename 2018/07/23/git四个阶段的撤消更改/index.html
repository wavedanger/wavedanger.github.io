  • 4个区

如上图所示，这里有四个区：

* 工作区
* 暂存区
* 本地仓库
* 远程仓库
  • 3个步骤

如上图所示，三个步骤对应三个区：

* git add
* git commit -m "message"
* git push
  • 5种状态
    • 未修改 ——原始状态
    • 已修改 ——已更改内容
    • 已暂存 ——git add
    • 已提交 ——git commit
    • 已推送 ——git push
